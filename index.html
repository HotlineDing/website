Custom built keyboard

I utilized tools such as AutoCAD, KiCAD, and a laser cutter to design and create the PCB and keyboard case. Amazon Product Tracker

Developed and deployed web application to periodically scrape products from NewEgg and Amazon for price data and product information and group them in categories. Then using Flask and python as a backend, the data would be stored in MongoDB and an SQL database to allow users to query and track product price over time and compared to others in its same category.

Realtime Chatroom

Using NodeJS with HTML and CSS I was able to create a realtime chat interface where users can type to multiple other users. Rooms can be created to seperate chat messages from other rooms. There is also a users list that notifies and updates whenever a user connects or disconnects. By integrating a MySQL database, I am able to query and log messages and keep a history of messages and users activity.

Raspberry Pi Keypad Lock

For this projecct I wrote an entire single-threaded assembly process to perform matrix scanning on an external keypad for a Rasberry Pi 4. This required me to learn ARMv8 assembly language and how to deadlock the uncessary cores, initalize my own stack, talk and interact with the GPIO pins, and more. Used additional toolchains and modified linkers to cross-compile for boot sector of the ARMv8 chip. Then used 8-segment display to output the keypresses and act as a keypad.
